# What is REST? ### REST stands for Representational state transfer. #### And it is a way for two machines to transfer the state of a resource via representations
# What is Data in REST? Data refers to all the RAW data we can access in from the database, or state or store

# What is a Representation in REST? In REST-speak, a client and server exchange representations of a resource, which reflect its current state or its desired state.
